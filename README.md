## Steps on how to create GitHub Page
1. Open the GitHub repository.
2. Click Settings.
3. Go to Pages in the sidebar.
4. Under Source, select the main branch.
5. Choose the /root folder.
6. Click Save.
7. Wait a few seconds for GitHub to generate the page.
8. Open the provided link to view the website.
